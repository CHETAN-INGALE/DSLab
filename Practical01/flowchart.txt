Start

Read the number of elements (n) for the array

Create an array of size n

Read the elements from the user and store them in the array

Display the menu of operations

Read the user's choice (choice)

If choice is 1:
Read the element to search (target)
Perform a linear search to find the element in the array
Display the result (found or not found) along with the index

If choice is 2:
Sort the array in ascending order using the Bubble Sort algorithm
Display the message "Array sorted successfully."

If choice is 3:
Reverse the array elements
Display the message "Array reversed successfully."

If choice is 4:
Display the elements of the array

If choice is 5:
Display the message "Exiting the program."
End

If choice is invalid (not 1, 2, 3, 4, or 5):
Display the message "Invalid choice. Please enter a valid option (1/2/3/4/5)."

Repeat from step 5
